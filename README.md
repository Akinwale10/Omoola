## ğŸš€ Getting Started

### Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/Akinwale10/Omoola.git
   cd Omoola
   ```

2. **Add your images**
   - Place your images in the `assets/images/` folder
   - See `assets/images/README.md` for the list of required images

3. **Update contact information**
   - Open `index.html`
   - Update phone numbers, email addresses, and physical address
   - Update WhatsApp number in the floating button

4. **Open in browser**
   - Simply open `index.html` in your web browser
   - Or use a local server:
     ```bash
     python3 -m http.server 8000
     # Visit http://localhost:8000
     ```

### Deployment

Upload all files to your web hosting service:
- **Shared Hosting**: Upload via FTP/cPanel
- **GitHub Pages**: Enable in repository settings
- **Netlify/Vercel**: Connect your repository for auto-deployment
